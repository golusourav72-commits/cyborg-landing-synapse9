# SYNAPSE-9 — Cyborg-Themed Landing Page

A responsive landing page built for the Techfest Campus Ambassador "Cyborg-Themed Landing Page Development" task.

The page is designed around a single concept: a fictional neural-augmentation lab where a diagnostic terminal watches a human body transition into an augmented one. The signature visual is a waveform that starts as an organic heartbeat and morphs into a circuit trace as it moves across the hero section.

## Features
- Boot-sequence hero animation (terminal-style typing effect)
- Animated SVG waveform signature (organic → synthetic)
- Augmentation modules grid (Ocular Relay, Cortex Bridge, Myofiber Lattice)
- Human vs. Augmented spec comparison table
- Scroll-reveal on module cards (IntersectionObserver)
- Fully responsive, keyboard-accessible, respects `prefers-reduced-motion`

## Tech Stack
- HTML5
- CSS3 (custom properties, animations, grid/flexbox)
- Vanilla JavaScript
